The project uses Kafka to scrape data from the Wikipedia page below, clean it, and transform it into a usable CSV file that serves as the basis for a Tableau dashboard.   

Wikipedia page : https://en.wikipedia.org/wiki/List_of_association_football_stadiums_by_capacity
Tableau Dashboard: https://public.tableau.com/app/profile/luke.shaffer/viz/FootballStadiumsoftheWorld/FootballStadiumsoftheWorld?publish=yes
